Centering with CSS

Centering Text

Centering text inside of a container is relatively straight forward. We can use text-align: center; to center text horizontally.

HTML

  <div>
    <p>Center this!</p>
  </div>

CSS

  p {
    text-align: center;
  }

Center this!

As simple as this may seem, there are still some caveats to consider. Specifically, using text-align: center; will only work if the element being styled has the width to center its content appropriately. If the element is only as wide as the content of the text, our text centering won't appear to do anything.

HTML

  <div>
    <p>Center this!</p>
  </div>

CSS

  p {
    text-align: center;
    width: fit-content;
  }

While the above example appears to be centered quite poorly, as far as the browser is concerned, it is centered. The issue here is not that the text is not centered, rather the element is only as wide as its text, so it looks the same when left, right, or center aligned. In situations where the width of an element is being restricted to the width of its content, other means can be used to center it.
